Для работы программы достаточно расположить клиентскую часть и API(серверную) 
по разным узлам(серверам)
в конфигурационном файле(config.php) клиентской указать адрес сервера API,
а на сервере параметры базы данных(логин, пароль, адрес и название базы данных, которое ещё не занято)
При первом запуске скрипт создаст базу и сделает первоначальное заполнение

После этого можно заполнять список исполнителей и создавать задачи
